项目一

iFuel系列油气合一管理系统

项目描述

目前全国有9万多座加油站，其中社会油站有5万座。 为加油站加气站提供一套完整的后台管理系统， 适用于单站或多站联网使用。 在该项目中参与需求评审，测试案例设计，测试执行，跟踪缺陷，测试报告。

工作内容

  1. iPos前台收银系统测试
  2. iMis后台管理系统测试
  3. iCard发卡系统测试
    • 账户管理 /充值 冻结 解冻 销户 撤销充值
    • 卡管理 / SAM 单位卡 个人卡 不记名卡 内部卡 卡充值 圈提 圈存
    • 客户服务 / 挂失 解挂 销卡 解灰
    • 报表打印 /
  4. iHost总部系统测试
  5. iWeb远程查询系统测试

项目二

基于TQC平台的CNG/LNG加油机和加气机

项目描述

主要是为中石化，中石油集团提供新一代更加稳定，防作弊系统更强的加油机。 也为社会油站解决整站问题，同时提供加气机与后台管理系统。 在该项目中参与需求评审，测试案例设计，测试执行，跟踪缺陷，测试报告。

工作内容

  1. 加油加气基本功能和加油限制功能的测试
  2. 通信异常和断电异常的测试
  3. 黑名单与白名单和价油品代码的测试
  4. iHost总部系统测试
  5. 系统菜单功能与打印功能的测试

其他工作

3. 配合工程师工作，使用Python+wxPython写个小工具，根据协议将数据发送给语音板芯片。
